# GoodHost

GoodHost е уеб платформа, която свързва **домакини** и **доброволци** в България. Домакините предлагат подслон, а доброволците могат да намерят място за престой, да маркират посещения и да оставят оценки.

## Функционалности

### Потребители
- **Два типа профили** — домакин (host) и доброволец (volunteer)
- Регистрация с имейл, парола и лични данни
- Верификация на самоличността чрез **Stripe Identity**
- Вход/изход и имейл известяване при всяко влизане
- Смяна на забравена парола чрез имейл линк (валиден 1 час)

### Домакини
- Профил с биография, локация, брой гости и до **20 снимки**
- Добавяне и изтриване на снимки от профила
- Рейтинг система — доброволците оценяват домакина от 1 до 5 звезди
- Значка **"Препоръчан"** за домакини с висок среден рейтинг

### Доброволци
- Разглеждане и търсене на домакини по име или локация
- Маркиране на посещение при домакин
- Получаване на имейл с линк за оценка след посещение
- Оставяне на оценка и коментар (еднократно на домакин)

### Имейли (Resend API)
- Потвърждение при регистрация
- Известяване при вход
- Покана за оценка на домакин
- Линк за смяна на парола

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Backend | Python / Flask |
| База данни | SQLite (разработка) / PostgreSQL (продукция) |
| Верификация | Stripe Identity |
| Имейли | Resend API |
| Сървър | Gunicorn |

## Инсталация

```bash
# Клонирай репото
git clone <repo-url>
cd GoodHost

# Създай виртуална среда
python -m venv .venv
.venv\Scripts\activate   # Windows
source .venv/bin/activate  # Linux/Mac

# Инсталирай зависимостите
pip install -r requirements.txt
```

## Конфигурация

Създай файл `.env` в основната директория:

```env
SECRET_KEY=твоя-секретен-ключ
STRIPE_SECRET_KEY=sk_...
STRIPE_PUBLISHABLE_KEY=pk_...
STRIPE_WEBHOOK_SECRET=whsec_...
RESEND_API_KEY=re_...
```

## Стартиране

```bash
python run.py
```

Приложението ще работи на `http://localhost:5000`.
