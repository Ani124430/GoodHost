# GoodHost

GoodHost е уеб платформа, която свързва **домакини** и **доброволци** в България. Домакините предлагат подслон, а доброволците могат да намерят място за престой, да заявят посещение, да го маркират и да оставят оценка.

## Функционалности

### Потребители
- **Два типа профили** — домакин (host) и доброволец (volunteer)
- Регистрация с имейл, парола и лични данни
- Верификация на самоличността чрез **Stripe Identity**
- Вход/изход с имейл известяване при всяко влизане
- Смяна на забравена парола чрез имейл линк (валиден 1 час)

### Домакини
- Профил с биография, локация, брой гости, социален профил и до **20 снимки**
- Поле "Търся помощ с..." — домакинът описва с какво му трябва съдействие
- Редактиране на биографията директно от профила
- Добавяне и изтриване на снимки (съхранени в **Cloudinary**)
- **Интерактивен календар** — маркиране на заети дни (единични или диапазон), премахване с клик
- Преглед и управление на заявки за посещение — одобряване или отказване с причина
- Рейтинг система — среден рейтинг от 1 до 5 звезди, видим в списъка
- Значка **"Препоръчан домакин"** при ≥ 5 отзива и среден рейтинг ≥ 4.5

### Доброволци
- Разглеждане и търсене на домакини по име или локация
- Индикатор за наличност — "Свободен" или "Свободен от дата"
- Преглед на **календара на домакина** — заети и свободни дни
- **Заявка за посещение** — избор на период, брой гости и съобщение до домакина
- Статус на заявката — чакаща / одобрена / отказана
- **Маркиране на изминало посещение** след одобрение
- Получаване на имейл с линк за оценка след маркирано посещение
- Оставяне на оценка (1–5 звезди) и коментар — еднократно на домакин
- Изтриване на собствена оценка
- Преглед на всички отзиви за домакин в модален прозорец

### AI чатбот асистент
- Вграден чатбот на всяка страница
- Работи с **Groq API (Llama 3.3 70b)** — отговаря на въпроси за платформата на български

### Имейли (Resend API)
- Потвърждение при регистрация
- Известяване при вход
- Известяване до домакина при нова заявка за посещение
- Известяване до доброволеца при одобрение или отказ
- Покана за оценка след маркирано посещение
- Линк за смяна на парола

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Backend | Python / Flask |
| База данни | SQLite (разработка) / PostgreSQL (продукция) |
| Снимки | Cloudinary |
| Верификация | Stripe Identity |
| Имейли | Resend API |
| AI чатбот | Groq API (Llama 3.3 70b) |
| Сървър | Gunicorn |

## Инсталация

```bash
# Клонирай репото
git clone <repo-url>
cd GoodHost

# Създай виртуална среда
python -m venv .venv
.venv\Scripts\activate   # Windows
source .venv/bin/activate  # Linux/Mac

# Инсталирай зависимостите
pip install -r requirements.txt
```

## Конфигурация

Създай файл `.env` в основната директория:

```env
SECRET_KEY=твоя-секретен-ключ

STRIPE_SECRET_KEY=sk_...
STRIPE_PUBLISHABLE_KEY=pk_...
STRIPE_WEBHOOK_SECRET=whsec_...

RESEND_API_KEY=re_...

GROQ_API_KEY=gsk_...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```

## Стартиране

```bash
python run.py
```

Приложението ще работи на `http://localhost:5000`.
